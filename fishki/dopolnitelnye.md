# Дополнительные

### Музыка прямо в игре

Благодаря аддону, встроенного в Plasmo Voice, можно воспроизводить и записывать любую музыку на пластинку.

{% hint style="info" %}
&#x20;/disc search <запрос> - поиск музыки
{% endhint %}

{% hint style="info" %}
&#x20;/disc burn - записать пластинку
{% endhint %}

{% hint style="info" %}
&#x20;/disc erase - стереть пластинку
{% endhint %}

### Кастомный чат

На нашем сервере есть самописный плагин для чата, который вы можете настроить. Пропишите команду **/chat** чтобы увидеть параметры чата

<figure><img src="../.gitbook/assets/изображение (41).png" alt=""><figcaption></figcaption></figure>

### Небольшие плагины

{% hint style="info" %}
&#x20;/sit /lay /crawl – сидение, лежание, ползание
{% endhint %}

{% hint style="info" %}
&#x20;/me /do /try /coin /roll – действие от персонажа, описание действия вокруг, попробовать, подбросить монетку и рандомное число от n до n.
{% endhint %}

{% hint style="info" %}
&#x20;/afk – переход в AFK. В табе появляется месяц
{% endhint %}

{% hint style="info" %}
&#x20;/groups – активация групп Plasmo Voice. Можно общаться на расстоянии
{% endhint %}

{% hint style="info" %}
where — достаточно написать это слово в чате (без /), чтобы вывести свои координаты в формате **\[530, 64, 1020 МП]**
{% endhint %}

{% hint style="info" %}
&#x20;/coords nether/overworld – выводит координаты обычного мира в Незере и наоборот
{% endhint %}

{% hint style="info" %}
&#x20;$ – калькулятор. Введя, к примеру, $150+150, в чате появится ответ: 300
{% endhint %}

{% hint style="info" %}
Выпадение голов. При каждом убийстве игрока с него падает голова
{% endhint %}

{% hint style="info" %}
Настоящая рыбалка. Вылов рыбы как в реальной жизни с доп. информацией
{% endhint %}

{% hint style="info" %}
Алиасы. Написав /display name set <ник> <алиас> и в чате у указанного игрока будет указываться тот ник, который вы указали.
{% endhint %}
