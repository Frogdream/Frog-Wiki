# Персонажи

На сервере есть РП-система персонажей. Вместо своего ника ты можешь\
выступать от лица придуманного персонажа — с собственным именем и описанием.

### Как открыть настройки персонажа?

Введи одну из команд:

{% hint style="info" %}
/character — открыть меню персонажа
{% endhint %}

{% hint style="info" %}
/ch — сокращённая версия той же команды
{% endhint %}

### Что можно настроить

* **Имя** - имя персонажа, которое будет отображаться в чате вместо ника
* **Скин** - скин который будет отображаться для других игроков. Выбрать скин можно на сайте namemc.

В ближайшем будущем будет добавлено Описание и Биография персонажа

### Несколько персонажей

У тебя может быть несколько персонажей одновременно — переключаться между\
ними можно через то же меню /character.

{% hint style="info" %}
Обращаем ваше внимание, что обычным игрокам доступно создание всего 3 персонажей, а игрокам с Fusion целых 9!
{% endhint %}

### Правила

* Имя персонажа не должно нарушать [правила сервера](https://frogdream.gitbook.io/frogdream-docs/pravila/pravila-servera) - оскорбительные и похожие на никнеймы других игроков имена запрещены
* В рамках РП скрывать личность за персонажем - нормально и разрешено
* Использовать персонажа для нарушения правил вне РП запрещено - администрация всегда знает, кто стоит за персонажем
