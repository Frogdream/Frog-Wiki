# Банкоматы

На нашем сервере реализована удобная финансовая система с использованием банкоматов. Банкоматы позволяют игрокам безопасно хранить свои сбережения, управлять балансом и переводить средства другим участникам сервера.

### Где найти банкоматы?

Каждый игрок нашего сервера может создать свой банкомат. Достаточно просто создать одну из следующих конструкций:<br>

| Тип           | Основание ( `blockType` )      | Кнопка ( `buttonType` ) | Люк ( `trapDoorType` ) | Верхний блок ( `upperBlockType` ) |
| ------------- | ------------------------------ | ----------------------- | ---------------------- | --------------------------------- |
| **OAK**       | Обтесанное дубовое бревно      | Дубовая кнопка          | Дубовый люк            | Керамика (обычная)                |
| **BIRCH**     | Обтесанное березовое бревно    | Березовая кнопка        | Березовый люк          | Белая керамика                    |
| **SPRUCE**    | Обтесанное еловое бревно       | Еловая кнопка           | Еловый люк             | Коричневая керамика               |
| **JUNGLE**    | Обтесанное тропическое бревно  | Тропическая кнопка      | Тропический люк        | Оранжевая керамика                |
| **ACACIA**    | Обтесанное акациевое бревно    | Акациевая кнопка        | Акациевый люк          | Оранжевая керамика                |
| **DARK\_OAK** | Обтесанное бревно темного дуба | Кнопка из темного дуба  | Люк из темного дуба    | Черная керамика                   |
| **MANGROVE**  | Обтесанное мангровое бревно    | Мангровая кнопка        | Мангровый люк          | Красная керамика                  |
| **CHERRY**    | Обтесанное вишневое бревно     | Вишневая кнопка         | Вишневый люк           | Розовая керамика                  |
| **BAMBOO**    | Бамбуковые доски               | Бамбуковая кнопка       | Бамбуковый люк         | Бамбуковый блок                   |
| **CRIMSON**   | Обтесанный багровый стебель    | Багровая кнопка         | Багровый люк           | Пурпурная керамика                |
| **WARPED**    | Обтесанный искаженный стебель  | Искаженная кнопка       | Искаженный люк         | Зеленая керамика                  |
| **SAND**      | Резной песчаник                | Кнопка из чернита       | Дубовый люк            | Желтая керамика                   |
| **MUD**       | Саманные кирпичи               | Каменная кнопка         | Тропический люк        | Тропическая древесина             |
