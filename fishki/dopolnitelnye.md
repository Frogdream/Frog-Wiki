# Дополнительные

### Музыка прямо в игре

Благодаря аддону, встроенного в Plasmo Voice, можно воспроизводить и записывать любую музыку на пластинку.

{% hint style="info" %}
&#x20;/disc search <запрос> - поиск музыки
{% endhint %}

{% hint style="info" %}
&#x20;/disc burn - записать пластинку
{% endhint %}

{% hint style="info" %}
&#x20;/disc erase - стереть пластинку
{% endhint %}

### Кастомный чат

На нашем сервере есть самописаный плагин на чат, который вы можете настроить. Пропишите команду **/chat** чтобы увидеть параметры чата

### Небольшие плагины

{% hint style="info" %}
&#x20;/sit /lay /crawl – сидение, лежание, ползание
{% endhint %}

{% hint style="info" %}
&#x20;/me /do /try /coin /roll – действие от персонажа, описание действия вокруг, попробовать, подбросить монетку и рандомное число от n до n.
{% endhint %}

{% hint style="info" %}
&#x20;/afk – переход в AFK. В табе появляется месяц
{% endhint %}

{% hint style="info" %}
&#x20;/groups – активация групп Plasmo Voice. Можно общаться на расстоянии
{% endhint %}

{% hint style="info" %}
&#x20;where – слово, не команда. Выводит координаты в формате **\[530, 64, 1020 МП]**
{% endhint %}

{% hint style="info" %}
&#x20;/cords nether/overworld – выводит координаты обычного мира в Незере и наоборот
{% endhint %}

{% hint style="info" %}
&#x20;$ – калькулятор. Введя, к примеру, $150+150, в чате появится ответ: 300
{% endhint %}

{% hint style="info" %}
Мячик. Чтобы его сделать, достаточно бросить снежок
{% endhint %}

{% hint style="info" %}
Выпадение голов. При каждом убийстве игрока с него падает голова
{% endhint %}

{% hint style="info" %}
Настоящая рыбалка. Вылов рыбы как в реальной жизни с доп. информацией
{% endhint %}

{% hint style="info" %}
Вымачивание сухого бетона в котле. Shift + ПКМ по котлу с бетоном в руке и готово!
{% endhint %}
